Caption font licenses
=====================

The caption font binaries in ../caption_fonts/ are third-party fonts,
redistributed here under their respective licenses (full text in this folder).

  caption-montserrat.ttf  Montserrat ExtraBold  SIL Open Font License 1.1  (OFL-montserrat.txt)
  caption-anton.ttf       Anton                 SIL Open Font License 1.1  (OFL-anton.txt)
  caption-jost.ttf        Jost ExtraBold        SIL Open Font License 1.1  (OFL-jost.txt)
  caption-inter.ttf       Inter SemiBold        SIL Open Font License 1.1  (OFL-inter.txt)

The same four families are also self-hosted for the web preview as .woff2 in
frontend/public/fonts/ (caption-*.woff2), registered via @font-face in
frontend/src/app/globals.css, so the DOM preview and the ffmpeg/libass burn use
the same typefaces.

Montserrat/Jost/Inter are instantiated as static single-weight TTFs from each
family's upstream variable font (fontTools.varLib.instancer), so libass sees
one unambiguous weight per family rather than resolving a default instance
from a variable font.
